MoFlow: multi-omic relay velocity로 chromatin-transcription dynamics 읽기

Nature Communications 2026 Hong, Lee & Kim RNA velocity + ATAC
Problem

기존 velocity model은 local regulatory dynamics를 충분히 잡지 못함

RNA-only
  • scVelo류 ODE model은 RNA kinetic rate와 latent time을 추정
  • branching system에서 single path assumption이 약점
  • chromatin accessibility라는 upstream context가 없음
Multi-omic ODE
  • MultiVelo는 chromatin을 넣지만 fixed gene class에 의존
  • gene-specific latent time이 non-canonical lag를 보정할 수 있음
  • cell-specific kinetic variability를 유연하게 잡기 어려움
Core Idea

MoFlow는 chromatin-aware relay velocity model

입력

각 cell과 gene의 chromatin accessibility c, unspliced RNA u, spliced RNA s.

학습

DNN이 cell-specific alpha_c, alpha, beta, gamma를 예측하고 local future neighbor와 velocity 방향을 맞춘다.

차이

global latent time 대신 local relay alignment를 쓰고, chromatin opening/closing 중 낮은 angular error를 선택한다.

Method Output

Discrete model class 대신 continuous regulatory score를 계산

m1

chromatin closing 중에도 unspliced RNA가 증가하는 delayed repression score.

m2

chromatin이 열려 있는데 RNA가 감소하는 asynchronous repression score.

RNA-on/off

vu, vs 부호로 coordinated activation/repression fraction을 계산.

핵심 비교

  • 4693 cells, 842 genes에서 MoFlow velocity stream이 known hierarchy와 가장 잘 맞음

  • MultiVelo는 GluN lineage 사이에 overly linear dependency를 만듦

주요 수치

CBDir: MoFlow 0.362, MultiVelo 0.211, scVelo 0.211, cellDancer -0.015.

Figure 2 Interpretation

MoFlow score는 MultiVelo model 1/2 logic을 재현

Model 1/2와의 연결

MultiVelo model 1 gene은 MoFlow m1 score가 높고, model 2 gene은 m2 score가 높다. 즉 fixed switching time 없이도 repression mode를 continuous하게 읽는다.

Biology

model 2 gene은 Cyc. Prog.에서 높고 RG로 갈수록 감소한다. cell cycle gene repression이 chromatin closure보다 먼저 시작될 수 있다는 해석과 맞는다.

  • PDGFRA, MAP3K1은 unspliced RNA production이 유지되지만 spliced RNA는 감소

MultiVelo gene-specific time

PDGFRA/MAP3K1의 negative lag가 사라지고 expected c -> u -> s 순서로 정렬된다.

MoFlow global relay time

Lag pattern

Mechanism 후보

MoFlow

RNA-only baselines

scVelo와 cellDancer는 downstream progenitor에서 HSC로 돌아가는 spurious backflow를 보임.

MultiVelo

overly linear flow로 natural branching pattern이 흐려지는 경향.

Limitations

강한 해석이 가능하지만 mechanism validation은 아직 부족

저자 명시 한계

  • long-range enhancer-promoter interaction 미모델링
  • transcriptional memory와 motif-level regulation 미포함
  • gene-wise inference라 pathway-level coordination에 제한

해석상 주의

  • velocity field는 UMAP embedding distortion에 영향받음
  • negative lag의 half-life 근거는 외부 NIH3T3 자료
  • chromatin state 선택은 lower angular error이지 direct measurement가 아님
Final Takeaways

MoFlow의 핵심 가치는 fixed time/class를 줄인 local multi-omic kinetics

Method

cellDancer식 relay velocity에 chromatin-aware kinetic parameter를 결합했다.

Biology

chromatin-dependent repression, chromatin-independent transcriptional boost, RNA export-linked lag를 구분한다.
